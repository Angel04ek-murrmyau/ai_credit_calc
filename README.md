# ai_credit_calc
Проектная работа кредитного калькулятора на основе ИИ.

# Цель: Создание веб-приложения на Flask, которое принимает данные клиента и выдаёт кредитный вердикт.
## Логика проекта
    1. Ввод данных клиента через веб-интерфейс (пол, возраст, семейное положение, и т.д.)
    2. Обработка данных через формулу кредитного риска.
    3. Обработка данных нейросетевой моделью (локальная нейросеть через roocode, или api GigaChat).
    4. Вердикт: кредит одобрен/отклонён, а также более подробное описание от нейросети.
    5. Создание Docker образа с проектом.
    6. Развертвыание приложения в Kubernetes с помощью minikube или kubeadm.
    7. Интеграция CI/CD с помощью Jenkins (стадии Build, Test, Deploy).
    8. Мониторинг и логирование с помощью Prometheus + Grafana + Grafana Loki.

## Технологии
    1. Python 3.14
    2. Flask
    3. RooCode c Qween/Gemma, GigaChat API
    4. Jenkins
    5. Prometheus + Grafana + Grafana Loki
    6. Minikube/Kubeadm
    7. SQLite/PostreSQL/Json для хранения данных о клиентах
    8. Docker
